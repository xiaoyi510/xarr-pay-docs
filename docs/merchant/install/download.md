# 下载安装包

## 方案一：授权中心下载（推荐）

前往授权中心下载最新版安装包：[https://auth.xarr.cn](https://auth.xarr.cn)

![XArr授权中心下载](/assets/images/auth_download.png)

## 方案二：网盘下载

前往售后群公告中的网盘进行下载，请按照修改时间下载最新版或者指定版本。

![下载网盘文件](/assets/images/merchant/merchant_pan_download.png)

::: warning 注意
- 请确保下载的安装包与服务器系统架构匹配（Linux amd64）
- 下载后建议校验文件完整性，避免传输过程中损坏
:::
