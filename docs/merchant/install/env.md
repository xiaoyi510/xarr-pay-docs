# 环境需求

## 运行环境

| 组件 | 要求 |
|------|------|
| Nginx | 必需，用于反向代理 |
| MySQL | 5.7 及以上版本 |
| 操作系统 | Linux（CentOS、Debian、Ubuntu） |

> 使用 Docker 部署则无需关注系统环境，参考 [Docker 安装](/merchant/install/docker)。

## 服务器配置

### 最低配置

| 项目 | 要求 |
|------|------|
| CPU | 1核 |
| 内存 | 1G |
| 带宽 | 1M |

### 建议配置

| 项目 | 要求 |
|------|------|
| CPU | 2核 |
| 内存 | 2G |
| 带宽 | 3M |

## 新手建议

使用 2H2G 或更高配置的服务器，安装 [宝塔面板](https://www.bt.cn)，通过 LNMP 一键安装运行环境，再参考 [宝塔安装教程](/merchant/install/bt) 部署系统。
