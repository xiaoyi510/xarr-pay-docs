# XArrPay 商户版帮助中心

## 产品概述

XArrPay 商户版是一款专业的聚合支付管理系统，满足个人站长和企业运营的需求。采用 Go 语言服务端开发，支持高并发；前端使用 Vue + ElementUI + Vite 构建，提供流畅的操作体验。

## 产品优势

| 特性 | 说明 |
|------|------|
| 🛠 全自研开发 | 非二开系统，代码质量有保障 |
| 🎨 清爽界面 | Vue + ElementUI 前端，操作简洁直观 |
| ⚡ 高性能 | Go 语言服务端，轻松应对高并发 |
| 💳 多通道支持 | 支付宝、微信、银联、易支付、V免签、京东等 |
| 🔌 易支付协议 | 采用通用易支付接口，接入简单 |
| 🔄 在线更新 | 程序支持在线更新，持续维护 |
| 🧩 插件系统 | 支持自定义插件开发，灵活扩展 |
| 📄 模板引擎 | 支持自定义首页、支付页、通知模板 |
