# 支付宝商家账单模式

通过定时获取商家账单并同步至本地，实现自动对账。需要在支付宝开放平台创建网站应用，并获取账单查询接口权限。

## 接入步骤

### 1. 创建网站应用

前往 [支付宝开放平台](https://open.alipay.com) → 控制台 → 我的应用 → 网页&移动应用 → 创建应用

### 2. 获取接口权限

在应用详情 → 产品绑定 中，申请开通「商家账单」相关能力，确保拥有 **获取账单列表** 的接口权限。

### 3. 配置加签方式

在应用详情 → 开发设置 → 接口加签方式：

- 使用 [支付宝密钥生成工具](https://opendocs.alipay.com/common/02kipk) 生成密钥对
- 将**应用公钥**上传至支付宝开放平台，获取**支付宝公钥**
- **应用私钥**填入 XArrPay 后台

## 配置说明

| 配置项 | 说明 |
|--------|------|
| 应用ID（AppID） | 支付宝开放平台创建的网站应用 AppID |
| 应用私钥 | 密钥工具生成的应用私钥 |
| 支付宝公钥 | 上传应用公钥后平台返回的支付宝公钥 |
