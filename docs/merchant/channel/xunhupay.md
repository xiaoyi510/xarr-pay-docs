# 虎皮椒支付配置

虎皮椒是一个免签约的聚合支付平台，无需企业资质即可接入支付宝和微信支付。

## 接入步骤

1. 注册并登录 [虎皮椒官网](https://www.xunhupay.com)
2. 在商户后台获取 **AppID** 和 **AppSecret**
3. 在 XArrPay 后台添加虎皮椒通道，填入以下信息

## 配置说明

| 配置项 | 说明 |
|--------|------|
| AppID | 虎皮椒商户后台的 AppID |
| AppSecret | 虎皮椒商户后台的 AppSecret |

::: tip 提示
虎皮椒支付宝和微信是分开的应用，需要分别获取对应的 AppID 和 AppSecret。
:::

## 支持的支付方式

- 支付宝
- 微信
