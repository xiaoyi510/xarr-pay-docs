# 易支付配置

易支付是一种通用的聚合支付接口协议，广泛应用于各类支付平台。通过对接易支付通道，可以快速接入第三方支付平台的支付能力。

![image](https://github.com/user-attachments/assets/89d7d255-3dbb-4507-bdea-4090c922290c)

## 配置说明

在 XArrPay 后台添加易支付通道时，需要填写以下信息：

| 配置项 | 说明 | 示例 |
|--------|------|------|
| 通讯地址 | 易支付接口入口地址 | `https://xxx.com/xpay/epay` |
| 商户ID | 易支付平台中分配的商户ID | `10000` |
| 通讯密钥 | 易支付平台中的通讯密钥 | 在易支付平台商户后台获取 |
| 请求方式 | 接口请求方式，一般选择 POST | `POST` |
| 接口类型 | 接口调用方式，见下方说明 | `mapi.php` |

## 接口类型说明

| 类型 | 说明 | 推荐 |
|------|------|------|
| `mapi.php` | 通过服务端接口返回二维码和跳转地址 | ✅ 推荐 |
| `submit.php` | 通过页面 form 表单提交创建订单 | - |

::: tip 建议
优先选择 `mapi.php` 接口类型，服务端直接交互更稳定，且支持获取二维码链接。
:::
