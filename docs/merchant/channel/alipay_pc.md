# 支付宝电脑网站支付

支付宝电脑网站支付适用于 PC 端网页场景，用户在电脑浏览器上打开商户网页，跳转支付宝页面扫码或登录账户完成付款。

## 所需资料

### 1. AppID

获取位置：[支付宝开放平台](https://open.alipay.com) → 控制台 → 我的应用 → 网页&移动应用

- 创建或选择已有应用，获取 **AppID**

### 2. 应用私钥和支付宝公钥

获取位置：应用详情 → 开发设置 → 接口加签方式

- 使用 [支付宝密钥生成工具](https://opendocs.alipay.com/common/02kipk) 生成密钥对
- 将**应用公钥**上传至支付宝开放平台，获取**支付宝公钥**
- **应用私钥**填入 XArrPay 后台

### 3. 开通产品

获取位置：应用详情 → 产品绑定

- 需开通「电脑网站支付」能力

## 配置说明

| 配置项 | 说明 |
|--------|------|
| AppID | 支付宝开放平台应用 AppID |
| 应用私钥 | 密钥工具生成的应用私钥 |
| 支付宝公钥 | 上传应用公钥后平台返回的支付宝公钥 |
